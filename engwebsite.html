Data Analysis and Processing

This is where the process starts. Data analysis and processing is one of the most important steps in machine learning. In essence, it's taking data and turning it into something useful. It starts with filling in any missing data because a machine learning model can't work properly with an incomplete dataset. Next, the data needs to be put into a form that the code will understand. This process can involve any combination of imputing, encoding, scaling, and more(cite steps of ml). We will cover some of these later on. Once this is done, our data processing is done. Data Processing for Business

Have you ever taken a survey about a product or yourself? Chances are, you have. Let's say a business wants to know more about it's employee salaries. They want to examine differences between pay for different genders and ages of their employees. Some employees may not want to share their age or gender, so the business needs to find a way to handle this missing data. Data processing can accomplish this by imputing the data. Now that they have all of the data filled in, they can use it to make calculations.

Data Processing with Java

Java is a very verbose language. This means it takes lots of code to do some things compared to other languages. Java also does not have many built in libraries for data processing which means there's a bit more manual work to do. Java makes up for this with it's security, multithreading capabilities, and fast runtimes (cite py vs java for ml). Each of these are important for a machine learning project, especially on a large scale like a business will have. Java accomplishes its security largely through the Java Virtual Machine. Code is compiled into bytecode which can be executed on any machine. Then, bytecode verification is performed before the program is executed to make sure the code is safe. The JVM and bytecode compilation are also big reasons for why Java is so fast. Java compilation does not consume processor or memory resources, which is a big advantage when using it for machine learning and an even bigger advantage for businesses who need to be very careful allocating their resources. (cite why java is secure)

The below example shows a simple process. In a list of employees, it finds the average salary of every employee over the age of 30. This is a very basic example of data processing. 


// Java Example: Finding the average salary of people over 30 years old
public class DataAnalysis {
    List<Person> persons = new ArrayList<>();
    try (CSVReader reader = new CSVReader(new FileReader(filepath))){
        String[] line;
        reader.readNext();
        while ((line = reader.readNext()) != null) {
            String name = line[0];
            int age = Integer.parseInt(line[1]);
            double salary = Double.parseDouble(line[2]);
            persons.add(new Person(name, age, salary));
        }
    } catch (CsvValidationException | IOException e){
        e.printStackTrace();
    }
    double avg = persons.stream().filter(person -> person.getAge() > 30)
                    .mapToDouble(Person::getSalary)
                    .average().orElse(0.0);
    System.out.printf("Average salary for people over 30: $%.2f%n", avg);
}

Data Processing with Python

Python is not nearly as verbose as Java. Throughout this article, this will be clear in each example. This also makes it very simple to write code for machine learning. It also has a diverse selection of libraries to help streamline this process further. While Python has advantages that make it easier to write code, it is not without faults. Python uses an interpreter to run its code which means the program is compiled line-by-line. This makes Python much slower than Java. Businesses must consider whether it is worth it to have faster runtimes or faster development times when chosing between languages. (cite py vs java for ml)
